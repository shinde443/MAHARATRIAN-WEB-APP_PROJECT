# Maharashtrian Recipe App

A simple and responsive web application built using HTML, CSS, and JavaScript. This project showcases popular Maharashtrian recipes with images and a search feature.

## Features

- Search recipes by name
- Display popular Maharashtrian recipes
- Recipe gallery with images
- Responsive design
- Simple and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
MAHARATRIAN-WEB-APP_PROJECT/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/shinde443/MAHARATRIAN-WEB-APP_PROJECT.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.
